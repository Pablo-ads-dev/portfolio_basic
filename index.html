API's que já apliquei profissionalmente:

Vindi Pagamentos

Desenvolvi a integração completa com a API da Vindi utilizando sua documentação oficial, implementando o fluxo de pagamentos da plataforma Fourpet. A solução contemplou autenticação, criação de cobranças, gerenciamento de transações e tratamento dos retornos da API, garantindo um processo de pagamento seguro e automatizado.

Focus NF-e

Implementei a integração com a API da Focus NF-e para automatizar o cadastro de empresas, emissão de NF-e, NFS-e (municipal e padrão nacional) e NFC-e. Também desenvolvi o consumo dos serviços da API, validação dos dados fiscais e tratamento das respostas para garantir a confiabilidade do processo de emissão.

ViaCEP

Integrei a API do ViaCEP para automatizar o preenchimento de endereços a partir do CEP informado pelo usuário. A implementação incluiu consumo da API, validação dos dados retornados, tratamento de CEPs inválidos e preenchimento automático dos campos de endereço, proporcionando uma experiência mais ágil e reduzindo erros de digitação.

Groq

Desenvolvi integrações com a API da Groq para incorporar funcionalidades de inteligência artificial em aplicações web. A implementação envolveu o envio de prompts, processamento das respostas do modelo, gerenciamento de contexto das conversas e tratamento de erros, possibilitando a automação de tarefas e a criação de assistentes inteligentes com respostas rápidas e eficientes.
